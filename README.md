# Ares: Local AI Agent Orchestrator

Ares is a stateful, autonomous AI agent framework designed for local execution. Built to move beyond simple chat interfaces, Ares uses an orchestration layer to give local Large Language Models (LLMs) the ability to reason, execute tools, and maintain semantic memory.

## Tech Stack
* **Orchestration:** LangGraph / LangChain
* **Inference Engine:** Local LLM (Ollama)
* **Environment:** Python 3

*This repository is under active development.*
